Путь к стартовой директории указываеться в файле path.ini:

Для Django: H:\Projects\Test\django_test\_test\path.ini.
Для Flask: H:\Projects\Test\flask_test\app\path.ini.

Приложение открывается по ссылке:

Для Django: http://127.0.0.1:8000/test
Для Flask: http://127.0.0.1:5000/test
